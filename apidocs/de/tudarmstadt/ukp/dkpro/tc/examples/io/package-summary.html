    BrownCorpusReader
    Reads the Brown corpus and sets POS tags for each tokens as classification outcome.

    MovieReviewCorpusReader
    Reads the Movie Review Corpus.
    NERDemoReader
    Reads the GermEval 2014 NER Shared Task dataset.
    PairTwentyNewsgroupsReader
    Reads pairs of TwentyNewsgroups corpus texts.
    ReutersCorpusReader
    Reads the classic Reuters text classification corpus
    SimpleDkproTCReader
    A very basic DKPro TC reader, which reads sentences from a text file and labels from another text file.
    STSReader
    Reads the Semantic Text Similarity (STS) SemEval format.
    TwentyNewsgroupsCorpusReader
    Reads the classical TwentyNewsgroups text classification corpus.
    UnlabeledTweetReader
    Reads plain text tweets, labels each tweet as sentence, and assigns no label to them.
