Class TwentyNewsgroupsUsingTCEvaluationDemo

  • java.lang.Object

    Constants

    public class TwentyNewsgroupsUsingTCEvaluationDemo
extends Object
implements Constants
    This a pure Java-based experiment setup of the TwentyNewsgroupsExperiment. Defining the parameters directly in this class makes on-the-fly changes more difficult when the experiment is run on a server. For these cases, the self-sufficient Groovy versions are more suitable, since their source code can be changed and then executed without pre-compilation.
