Class SparseFeatureStore

  • java.lang.Object
    • de.tudarmstadt.ukp.dkpro.tc.fstore.simple.SparseFeatureStore
  • All Implemented Interfaces:
    FeatureStore

    public class SparseFeatureStore
extends Object
implements FeatureStore
    Feature store that internally uses hash maps instead of arrays to store sparse features. All instances retrieved from this FeatureStore by getInstance(int) have the same number and ordering of features, but some features might have null value.
