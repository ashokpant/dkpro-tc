      • passesNgramFilter

        public static boolean passesNgramFilter(List<String> tokenList,
                        Set<String> stopwords,
                        boolean filterPartialMatches)
        An ngram (represented by the list of tokens) does not pass the stopword filter: a) filterPartialMatches=true - if it contains any stopwords b) filterPartialMatches=false - if it entirely consists of stopwords
        Parameters:
        tokenList - The list of tokens in a single ngram
        stopwords - The set of stopwords used for filtering
        filterPartialMatches - Whether ngrams where only parts are stopwords should also be filtered. For example, "United States of America" would be filtered, as it contains the stopword "of".
        Returns:
        Whether the ngram (represented by the list of tokens) passes the stopword filter or not.
