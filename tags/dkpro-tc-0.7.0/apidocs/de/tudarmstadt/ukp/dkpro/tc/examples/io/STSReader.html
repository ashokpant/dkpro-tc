      • getBaseUri

        protected String getBaseUri()
        Description copied from class: AbstractPairReader
        If the pair of texts come from one document, set this to the document's original pathname, without the final filename. It is used if the CAS is written to file. The pathname does not need to actually exist.

        The docID, combined with the BaseURI should be unique for each instance.

        Note: If each of your data instances doesn't have its own source file, then we recommend using an empty string here, or a very simple directory name. Otherwise, DKPro Lab may lose some of your data without loudly notifying you.

        To check for lost data, look in "Evaluation[...]" file and add up the experiment's "Correctly Classified Examples" and "Incorrectly Classified Examples".
